simetbox-openwrt-base
---------------------

SIMET1 has been decomissioned, and the three scripts in this package
that still had any use were moved to simetbox-openwrt-feed.

The source code that existed here is in branch "simet1", for historic
purposes AND to ensure license compliance.

To avoid confusion and doubt, the master branch is now empty but for a
README.md with this information.

For the replacement for SIMET1, please refer to the "simet-ma" package
for the SIMET2 measurement engine, and to the simetbox-openwrt-feed
OpenWRT feed for the OpenWRT integration.
